# astrbot_plugin_group_vibe

一个 AstrBot 群聊氛围插件，用来让 QQAssistant 在群聊里像普通群友一样低频、看氛围、看上下文地自然接话。

## 功能

- 只监听群聊消息，可按群号白名单限制生效范围。
- 根据群聊活跃程度动态调整接话冷却。
- 为普通消息、疑问句、气氛词、连续互动设置不同回复概率。
- 有防刷屏阈值，避免短时间内连续刷屏。
- 支持热点素材辅助聊天：低频读取 60s / DailyHub 同源热榜，只作为找话题素材，不会直接播报新闻。
- 支持轻量自动真假判断：群里出现“真的假的、网传、有出处吗”等明显信号时，低频顺手判断或提醒看来源。
- 支持 AstrBot WebUI 插件设置页调整参数。

## 推荐额外插件

本插件可以单独运行；下面这些插件用于补齐“自主聊天”的完整能力。

| 插件 | 推荐用途 |
| --- | --- |
| `AMag1c/astrbot_plugin_dailyhub` | 每日资讯、热搜、AI 日报、Epic 免费游戏等信息源。建议把每日推送设置为私聊 QQ，不要默认发群。`group_vibe` 可读取同源热点作为自然聊天素材。 |
| `konley/astrbot_plugin_isittrue` | 手动事实核查插件。群友可以通过 @ 或引用消息触发更明确的“是真的吗”判断。`group_vibe` 内置的是低频轻量判断，用于无感知对话。 |
| `astrbot_plugin_anysearch`（可选） | 给事实核查提供联网搜索增强。未安装时，“是真的吗”和 `group_vibe` 仍可用，但会更依赖模型自身判断。 |

## 推荐默认参数

- 冷群冷却：60 秒
- 中等活跃冷却：25 秒
- 很活跃冷却：10 秒
- 连续互动间隔：3 秒
- 普通消息概率：0.22
- 疑问句概率：0.65
- 气氛词概率：0.45
- 连续互动概率：0.90 到 0.96
- 热点素材附带概率：0.18
- 自动真假判断概率：0.38
- 自动真假判断冷却：480 秒
- 回复最大长度：45 字

## 安装

将本仓库放入 AstrBot 插件目录：

```text
data/plugins/astrbot_plugin_group_vibe
```

然后在 AstrBot WebUI 中重启或重载插件。

## DailyHub 配合方式

建议安装 `astrbot_plugin_dailyhub` 后，在其设置页使用“QQAssistant 简化推送目标”区域：

- `QQ 平台 ID`：当前通常填 `qq-test`
- `私聊推送 QQ 号`：填写需要接收每日资讯的人，例如 `2364241525`
- `群聊推送群号`：默认留空，避免资讯直接刷群
- `简化目标应用到哪些资讯源`：默认 `news60s,ai,epic`

这样每日资讯会私聊发给指定 QQ，而 `group_vibe` 只把热点作为聊天素材使用。

## 注意

这个插件适合小范围群聊氛围实验。建议明确让群友知道这是实验小号，并避免让机器人参与借钱、隐私、账号、承诺、争吵、感情决定等高风险话题。
