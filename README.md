# astrbot_plugin_group_vibe

一个 AstrBot 群聊氛围插件，用来让机器人在群聊里像普通群友一样低频、按气氛主动接话。

## 功能

- 只监听群聊消息。
- 可按群号白名单限制生效范围。
- 根据群聊活跃程度动态调整接话冷却。
- 对普通消息、疑问句、气氛词设置不同回复概率。
- 对连续互动场景使用更短间隔和更高回复概率。
- 有防刷屏阈值，避免短时间内连续刷屏。
- 支持 AstrBot WebUI 插件设置页调整参数。

## 推荐默认参数

- 冷群冷却：60 秒
- 中等活跃冷却：25 秒
- 很活跃冷却：10 秒
- 连续互动间隔：3 秒
- 普通消息概率：0.22
- 疑问句概率：0.65
- 气氛词概率：0.45
- 连续互动概率：0.90 到 0.96
- 回复最大长度：45 字

## 安装

将本仓库放入 AstrBot 的插件目录：

```text
data/plugins/astrbot_plugin_group_vibe
```

然后在 AstrBot WebUI 中重启或重载插件。

## 配置

插件提供 `_conf_schema.json`，安装后可在 AstrBot WebUI 的插件设置页调整：

- 是否启用插件
- 生效群号
- 冷却时间
- 回复概率
- 防刷屏阈值
- 模型温度
- 关键词列表

## 注意

这个插件适合小范围群聊氛围实验。建议先在明确知道机器人存在的测试群中使用，并避免让机器人参与敏感话题、隐私、借钱、争吵等场景。
